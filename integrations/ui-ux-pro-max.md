# UI/UX Pro Max

Instalación oficial mediante CLI:

```powershell
npm install -g uipro-cli
```

Dentro del proyecto:

```powershell
uipro init --ai codex
uipro init --ai claude
uipro init --ai cursor
```

Úsalo para:

- diseño de pantallas;
- generación de sistemas visuales;
- dashboards;
- formularios;
- accesibilidad;
- consistencia de componentes;
- revisión de anti-patrones UI/UX.

Las reglas funcionales, permisos, validaciones y contratos siguen perteneciendo al skill principal y a `AGENTS.md`.
