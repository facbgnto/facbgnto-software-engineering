# Integraciones opcionales

## Orden recomendado

1. `facbgnto-software-engineering`
2. `AGENTS.md` específico del proyecto
3. Graphify para navegación del código
4. LightRAG para conocimiento documental
5. UI/UX Pro Max para frontend
6. Superpowers para procesos de ingeniería
7. Everything Claude Code ZH solo para capacidades seleccionadas

No actives múltiples reglas equivalentes. El objetivo es complementar, no acumular instrucciones.
