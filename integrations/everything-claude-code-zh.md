# Everything Claude Code ZH

Este repositorio contiene agentes, commands, hooks, rules, MCPs y skills traducidos al chino.

Se integra de forma **selectiva** porque puede solaparse con:

- Superpowers;
- Agent Skills;
- `AGENTS.md`;
- `facbgnto-software-engineering`.

## Descarga

```powershell
git clone https://github.com/xu-xiang/everything-claude-code-zh.git
```

## Componentes recomendados para evaluar

- patrones de TypeScript;
- patrones de Python;
- patrones de Java/Spring Boot;
- PostgreSQL;
- diseño API;
- migraciones;
- Docker;
- pruebas E2E;
- revisión de seguridad;
- bucle de verificación.

## Regla

No copies el repositorio completo dentro de cada proyecto. Selecciona solo los skills o reglas que falten y revisa el idioma, rutas, hooks y comandos antes de activarlos.
