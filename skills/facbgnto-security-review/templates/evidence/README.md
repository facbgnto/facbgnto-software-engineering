# Evidencias de seguridad

Guarda únicamente evidencia sanitizada.

```text
evidence/
├── commands/
├── sanitized-output/
├── diagrams/
└── screenshots/
```

No almacenes:

- secretos;
- tokens completos;
- contraseñas;
- datos personales;
- archivos `.env`;
- respaldos;
- información productiva sensible.
